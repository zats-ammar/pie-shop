# Welcome to project 'bethonys pie-shop'!!!

## What made me to build this project?

This is my first MVC project using ASP.NET core 2.2, as to learn the core concepts of building a mvc app with latest asp.net core version. 

## What's this app about?

This is a simple app built for Bethony to manage her pie-shop by listing products and view their details. Also it allows authenticated users to send feedbacks to bethany about her products. 

## Main technologies used

[ASP.NET Core v2.2](https://docs.microsoft.com/en-us/aspnet/core/?view=aspnetcore-2.2)
> A cross-platform, high-performance, open-source framework for building modern, cloud-based, Internet-connected applications.

[EF Core](https://docs.microsoft.com/en-us/ef/core/)
> A lightweight, extensible, open source and cross-platform version of the popular Entity Framework data access technology.
